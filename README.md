# MANA Yerba Mate – Interactive 3D eCommerce Website

## Overview

MANA Yerba Mate is a highly interactive 3D eCommerce platform designed to deliver a unique product discovery and online shopping experience.

The project combines immersive 3D product visualization, advanced animations, dynamic interactions, and modern eCommerce functionality to create an engaging digital experience for customers.

**Live Website:** https://manayerbamate.com/
